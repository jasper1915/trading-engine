# 📈 Stockify: High-Performance Multi-Asset Exchange

Stockify is a professional-grade, full-stack trading platform designed for high-speed order matching and real-time portfolio analytics. Built with a modern tech stack, it features a glassmorphic dark-themed UI and an industry-standard matching engine.

## 🚀 Key Features
- **Multi-Asset Trading:** Support for BTC, ETH, SOL, XRP, ADA, DOGE, and DOT.
- **Matching Engine:** High-performance engine supporting Limit, Market, and Stop-Loss orders.
- **Portfolio Analytics:** Real-time PnL tracking with Weighted Average Cost (WAC) calculations.
- **Security Suite:** JWT-based authentication, 2FA support, and encrypted user profiles.
- **AI Strategist:** Integrated AI assistant for market insights and portfolio analysis.
- **Responsive Design:** Fully optimized for Desktop and Mobile devices.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Lucide-React, CSS (Glassmorphism)
- **Backend:** Spring Boot, Spring Security, JWT, PostgreSQL
- **AI Intelligence:** Google Gemini AI

## 📦 Project Structure
- `/engine`: Spring Boot backend services and the Matching Engine.
- `/frontend`: React-based trading dashboard and user interface.

## 🚦 Getting Started

### Prerequisites
- JDK 17+
- Node.js 18+
- PostgreSQL

### 1. Start the Backend
```bash
cd engine
./mvnw spring-boot:run
```

### 2. Start the Frontend
```bash
cd frontend
npm install
npm run dev
```
